Share Your Screen

To allow someone to control this computer remotely, you need to run the Host Agent on this machine.

Steps

  1. Install Node.js if you don't have it.
  2. Download or create the file host-agent.js with the code provided in your project.
  3. Open a terminal / command prompt in that folder.
  4. Run: 
    npm init -y
npm install ws robotjs
  5. Start the agent: 
    node host-agent.js
  6. It will show a 6-digit code in the terminal. Share that code with the person who will control your computer.
  7. The controller will enter that code on the “Control a Remote Computer” page.

You can stop remote control anytime by closing the terminal running host-agent.js.
